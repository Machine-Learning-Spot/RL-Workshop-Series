# Part I: Introduction to Reinforcement Learning

## Implementation
Algorithms:
1. OpenAI Gym Env 
2. Monte Carlo for FrozenLake Env
3. Temporal Difference for Taxi Env
4. Deep Q-Network for CartPole Env
5. GA and PSO algorithms for Battleship Env
